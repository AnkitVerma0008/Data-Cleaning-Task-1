# Task 1: Data Cleaning and Preprocessing

## Objective

The objective of this project is to clean and preprocess the Marketing Campaign dataset using Python and Pandas.

## Dataset

* Marketing Campaign Dataset

## Tools Used

* Python
* Pandas
* Visual Studio Code

## Steps Performed

* Loaded the dataset.
* Checked dataset information.
* Identified missing values.
* Filled missing values in the Income column using the mean value.
* Checked and removed duplicate records.
* Converted column names to lowercase.
* Converted the `dt_customer` column to datetime format.
* Verified data types.
* Saved the cleaned dataset.

## Files Included

* `marketing_campaign.csv` (Original Dataset)
* `cleaned_marketing_campaign.csv` (Cleaned Dataset)
* `data_cleaning.py` (Python Code)
* `README.md`

## Output

A cleaned dataset named `cleaned_marketing_campaign.csv` was generated successfully.

## Conclusion

The dataset was successfully cleaned and prepared for further data analysis.
